# 문제 10. [서술형] DRF CRUD에서 사용하는 HTTP method 종류 4가지와 각각 어떤 목적으로 사용하는지 작성하시오.
`GET`은 data를 조회하는데 사용되고, `POST`는 data 생성, `PUT`은 data 수정, `DELETE`는 data 삭제 목적으로 이용한다.
